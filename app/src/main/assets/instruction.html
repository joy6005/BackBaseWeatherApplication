Instructions

Overview

The application enables user to access the desired city weather information. Major cities are provided but user can search specific city from the search bar or pick location from the map. Application will also provide detailed weather information like Humidity, Pressure, Min/Max Temperature, Wind speed, cloud percentage.

Home Screen

The Home screen contains the list of popular cities.Users are able to select the any city right away from the list to get the weather information.

  • Search: User can search the city within the city list.
  • Add New: User can add new city by clicking the circular button at bottom left corner. User can directily type the city name or search from the map to pick the desired location.
  • Settings: Unit change option is provided to chage the unit (Metric/Imperial)
    User can also reset all cities from the bookmarked list.
  • Delete: To delete any city from the bookmark list, User just need to long press on perticular city and option to remove city will be shown.

City Screen

Once user select any city from the city list, He/She will able to see the weather details of that perticular details. Temperature, Pressure, Wind speed, CLoud, Min/Max Temperature, Humidity are shown for the perticular city.

User are also able to see the forecast of upcoming 5 days in the list below the current date weather. User can also refresh the weather details by clicking the refresh button.
